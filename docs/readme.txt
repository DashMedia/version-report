VersionReport
============

Tracks modx version numbers within Google Cloud Datastore


Installation
============

Install vis extras.io

Setup
=====

Once installed, you will need to add the contents of your Google API credentials JSON file to the `versionreport.api_key` system setting you can get a new JSON file from `https://console.cloud.google.com/apis/credentials?project=what-does-the-drone-say` (note that you can not re-download a json file, so keep it safe)
